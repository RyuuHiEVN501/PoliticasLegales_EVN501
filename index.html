Plazo de envío y gastos

1Envío estándar

Es el tipo de plazo de entrega que, por defecto, ofrecemos. Si el producto es empaquetado, el plazo de entrega está determinado para efectuarse en 30 días. Si el producto es perecedero, la entrega se efectuará en 7 días hábiles. El servicio estándar está disponible con todas las compras.




2Envío exprés o urgente

Si eres un cliente exigente, este plazo de entrega es para ti. Es el plazo de entrega que le recomendamos a nuestros clientes. Los productos empaquetados tienen un tiempo estimado de entrega de 7 días hábiles. Nuestros productos perecederos se entregan entre 1 y 3 días. Tiene un costo extra y está disponible para todas las compras.




3Envíos nacionales/internacionales

Unicamente nuestros productos premium internacionales están disponibles con este plazo de envío. Los gastos son mayores para mejorar la experiencia y la calidad del servicio. El gasto varía dependiendo de la zona del producto y el embalaje.

Política de devoluciones

1Excepciones comunes

  • Los artículos con descuento son definitivos y no se pueden devolver ni cambiar.
  • Los artículos devueltos deben tener las etiquetas aún puestas y deben devolverse en el embalaje original del producto.
  • Los artículos devueltos no deben tener signos visibles de desgaste o uso.




2Para iniciar una devolución, complete los siguientes pasos:

  1. Responda al correo electrónico de confirmación de su pedido para solicitar qué productos desea devolver
  2. Imprima la etiqueta de envío de devolución prepaga que recibirá por correo electrónico
  3. Envíenos todos los artículos usando la etiqueta provista




3Información adicional:

  • El reembolso tomará entre 1 o 3 meses para completarse satisfactoriamente.
  • El cliente debe pagar la tarifa de envío correspondiente al nuevo producto enviado.
  • En caso de robo o extravío, el cliente debe pagar una cantidad extra fija para asegurar la calidad de servicio.
